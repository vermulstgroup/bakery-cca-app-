# Review 6: UI Designer Visual Consistency

## CRITICAL Issues

1. **Margin color coding inconsistent** - `src/app/(main)/dashboard/page.tsx:160-170`
   - Uses text-green-200, text-amber-200, text-red-200
   - Other pages use text-green-400, text-amber-400, text-red-400
   - Creates visual inconsistency

## HIGH Issues

1. **Card border inconsistency**
   - Some cards use `border-slate-700`
   - Others omit borders entirely
   - No consistent wrapper component

2. **Button style mixing**
   - Button component has variants: default, destructive, outline, etc.
   - But pages use custom styled buttons like `bg-slate-700 hover:bg-slate-700`
   - Entry page quick-add buttons don't use component

3. **Spacing not on 8px grid**
   - Dashboard uses gap-3 and gap-2 interchangeably
   - Some cards use p-4, others p-6
   - Entry page has py-2 and py-3 mixed

4. **Typography hierarchy missing**
   - No consistent heading structure
   - Dashboard: text-2xl
   - Entry: text-xl
   - Summary: text-xl
   - No semantic meaning

5. **Rounded corners inconsistent**
   - Mix of rounded-xl, rounded-lg, rounded-md
   - Card component defaults to rounded-xl
   - Some inline cards use rounded-lg

## MEDIUM Issues

1. **Status color palette inconsistent** - `src/app/(main)/supervisor/page.tsx:91-98`
   - Status badges: green-500/20, red-500/20, yellow-500/20
   - Text indicators: green-400, red-400, amber-400
   - Colors don't match

2. **Empty states not designed**
   - Dashboard shows generic AlertTriangle
   - Different empty state patterns per page
   - No reusable empty state component

3. **Error/warning states inconsistent** - `src/app/(main)/settings/page.tsx:47-51`
   - Alert uses primary/10 background
   - Other pages use red-500/10
   - No standard error styling

4. **Skeleton loading UI missing** - `src/app/(main)/settings/edit-prices/page.tsx`
   - PriceSkeleton defined but not used elsewhere
   - Dashboard, entry, trends use Loader2 spinner

5. **Icon sizing inconsistent**
   - Emoji: text-3xl in cards, text-2xl in lists
   - Lucide: h-5/w-5, h-4/w-4, h-6/w-6 mixed

## LOW Issues

1. **Backdrop blur opacity varies**
   - Some: backdrop-blur-sm
   - Others: backdrop-blur-lg
   - No clear hierarchy rationale

2. **Text color naming mixed**
   - text-slate-400 vs text-muted-foreground
   - No consistent approach

3. **Gradient backgrounds not reusable**
   - from-slate-900 via-slate-800 to-slate-900 repeated
   - Should be component or class
