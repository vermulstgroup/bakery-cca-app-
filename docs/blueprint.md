# **App Name**: BISS Bakery App

## Core Features:

- Daily Production Entry: Record daily production quantities for each product.
- Sales Tracking: Track daily sales for each product, updating revenue calculations automatically.
- Expense Management: Log weekly expenses by category, with quick-add amount suggestions.
- Profit Calculation: Automatically calculate weekly and monthly profit based on sales and expense data.
- Real-time Reporting: Generate reports on production, sales, expenses, and profitability with data visualizations.
- Offline Data Sync: Enable offline data entry and automatic synchronization when internet connection is restored.
- Trend analysis: Analyze sales data from previous weeks to provide personalized offers

## Style Guidelines:

- Primary color: Teal (#0D9488), with a gradient to #14B8A6 for a sophisticated and trustworthy feel.
- Background color: Warm off-white (#FAFAF9) in light mode, near black (#1C1917) in dark mode, to ensure legibility in various lighting conditions.
- Accent color: Amber (#F59E0B) for CTAs and quick-add buttons, providing clear visual cues for bakers with flour-covered hands.
- Display and headline font: 'Plus Jakarta Sans', sans-serif, for headings and display.
- Numbers and currency font: 'DM Mono', monospace, for clear presentation of numerical data.
- Note: currently only Google Fonts are supported.
- Minimum button height: 56px to accommodate users with flour-covered hands.
- Spacing: Consistent spacing system (4px, 8px, 12px, 16px, 24px, 32px) for a balanced and harmonious design.
- Subtle animations: Micro-interactions such as button presses and number count-ups to provide clear feedback and enhance user experience.
- Use simple bakery-related icons and emojis.
